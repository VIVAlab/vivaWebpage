Single-view obstacle detection for smart back-up camera systems

Objective

To develop a driver-assistance backup system that uses rear-view camera in order to perform obstacle detection during reverse drive manoeuvres.

Outcome

A combined camera and range sensor in a single, low-cost, piece of hardware.

Difficulties

Run in real-time;
Simple enough to be ported to a dedicated image processing chip;
Perform reliably to be released as a consumer product.
